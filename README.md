### PROJETO DE ESTUDO EXPRESS 
----

### MAIS SOBRE O PROJETO 

Este projeto foca principalmente no estudo do framework Express mas também tecnologias que, em conjunto com o Express, proporcionam um aprofundamento mais abrangente sobre o mesmo. A abordagem que utilizo é sempre visando boas práticas que o Express tem a oferecer, utilizando os recursos básicos do Express para a comunicação com o banco de dados relacional Postgres.
Basicamente o sistema implementa uma estrutura de classes que permite estabelecer relacionamentos entre salas, aulas e vídeos. O objetivo é manter um paralelismo próximo ao contexto do mundo real. Assim, é possível criar salas e, dentro delas, cadastrar aulas/disciplinas que serão ministradas na respectiva sala. Os vídeos representam as aulas diárias.

---
### TECNOLOGIAS USADAS

- Nodejs
- Expressjs
- PostgresSQL
- Docker (criação do container postgres)
- TypeORM
